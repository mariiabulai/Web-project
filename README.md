Навчальний веб-проєкт, створений у рамках лабораторної роботи з дисципліни "Веб-орієнтована розробка ПЗ".

## Опис проєкту

**Fitness&Health** — онлайн-платформа для відслідковування здорового способу життя. В ній можна робити записи тренувань, створювати персоналізовані плани харчування та щоденні завдання. Сервіс допомагає користувачам відслідковувати прогрес, формувати правильні звички та отримувати мотивацію.

## Структура проєкту

Проєкт має наступну файлову структуру:

`````

/fitness-health-project
    ├── /images           # Зберігання зображень (логотип, тощо)
    ├── /css              # Зберігання файлів стилів (CSS)
    ├── /js               # Зберігання скриптів (JavaScript)
    ├── index.html        # Головна сторінка
    ├── about.html        # Сторінка "Про сервіс"
    ├── contacts.html     # Сторінка "Контакти"
    ├── login.html        # Сторінка "Реєстрація/Вхід"
    └── dashboard.html    # Особистий кабінет користувача
`````
